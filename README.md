# Velovita WordPress Technical Test

## Description
Custom WordPress theme built with PHP following Velovita brand guidelines.

## Requirements
- PHP 8.x
- WordPress 6.x
- Local

## Installation
1. Install WordPress locally.
2. Copy the `gara-theme` folder into `wp-content/themes`.
3. Activate the theme from the WordPress admin panel.

## Features
- Custom WordPress theme (no builders)
- Responsive layout
- Custom Post Type: Products
- Contact form

## Notes
Project developed focusing on clean structure, WordPress best practices, and brand consistency within the given time constraint.
